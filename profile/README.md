# 《曙光宣言》

1. 打破謊言，終結暴政！
1. 沒有特權才有新國度；自由不是恩賜，要親手奪回！
1. 起來吧，不願做奴隸的人們，起來捍衛屬於每個人的權利！
1. 要真相不要虛假，要自由不要恐懼；讓掌權者歸於人民的監督！

我們並非為了仇恨而發聲，而是為了人的尊嚴。歷史告訴我們，任何遮蔽真相的權力終將腐朽；任何踐踏自由的制度都會在沉默中裂開。今天，我們選擇不再沉默，因為沉默是暴政最廉價的燃料。

## 我們宣告：

人的生命、良知與自由不屬於任何黨派、階層或利益集團。言論與結社的自由不是施捨的恩典，而是出生即被賦予的權利；權力不是神授，而是人民授權並可隨時收回的責任。任何以安全之名行箝制之實、以繁榮之名行掠奪之實的統治，都是對公民的侮辱。

## 我們主張：

— 以法治代替人治，讓法律成為護衛者而非鎖鏈；
— 以選票替代恐懼，讓權力在週期中接受更替與審計；
— 以真相對抗宣傳，讓獨立媒體與公開資訊還原事實；
— 以獨立司法守護公平，讓每一個人都能在法庭前平等；
— 以地方自治分散權力，讓公共資源回到社區與公民；
— 以社會連帶扶持弱勢，讓文明的進步不再是少數人的特權。

## 我們承諾：

我們將以非暴力的方式推動改良，以公開透明的方式建立信任，以團結互助的方式抵禦恐嚇。我們拒絕將仇恨作為動員的藥引，也拒絕以謊言換取短暫的掌聲。當有人以恐懼威脅我們的沉默，請記得：恐懼只會擴張，直到有人說出第一句真話；而今天，真話已經說出。

## 我們相信：

真正的強大不是來自豐收的口號，而是來自誠實的制度與自由的人民。當這片土地終於擁抱民主與法治，它將不再以武力震懾鄰里，而會以開放、信任與合作成為周邊諸邦的樞紐；不是以征服建立秩序，而是以規則、契約與尊重凝聚共同體。自由社會的吸引力，將勝過任何恐懼；可信制度的穩定，將超越任何威權的短暫繁榮。

## 我們呼喚：

每一位仍在沉默的人，請守護你心中的微光；每一位仍在猶豫的人，請相信你並不孤單。把恐懼留在昨日，把勇氣帶到今日：在學校說真話，在工廠守規則，在社區護公平，在網路傳遞可核實的資訊，在公共領域以文明辯論改變共識。當千萬盞燈同時點亮，黑夜便無處可棲。

## 最後，我們願以此宣言相勉：

我們要求真相，因為真相使人自由；
我們要求自由，因為自由使人負責；
我們要求責任，因為責任使權力謙卑。
願每個人都成為自己的火光，照亮他人，也照亮未來。
